
# Korean-English Weather App 🌤️
Live demo: [https://sweety-hjh223.github.io/toggle-weather/]

A bilingual weather application supporting 
English and Korean languages.

Built with: HTML, CSS, JavaScript, OpenWeather API

Features:
- Real time weather data
- Full Korean/English language toggle
- Responsive design for all devices
- 5 day forecast
